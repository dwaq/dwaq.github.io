    • An open source, internet enabled home automation project that allows any web browser to control and read the state of digital devices

    • A modern replacement of an Atari controller that transforms between a joystick and a paddle using the same base

    • A self-contained, smart tester that can determine the characteristics of a coil such as a relay

  • Particle Mesh Home Climate Monitoring
    • 10x Particle Mesh devices with a custom sensor shield monitors temperature, humidity, CO2, and TVOC all over my house
  • Artemis Says: A voice-controlled Simon clone with Tensorflow
    • A clone of the classic game "Simon" utilizing a Sparkfun Artemis board with machine learning to control the game using only voice commands
